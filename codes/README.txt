Can download SimpleNLG (for converting phrases to sentences) from : http://code.google.com/p/simplenlg/downloads/list
(Tutorial of how to use SimpleNLG available at : http://code.google.com/p/simplenlg/wiki/Tutorial)

Can download Lexicon used from : http://code.google.com/p/simplenlg/wiki/AppendixC

Can download Stanford CoreNLP toolkit from : http://nlp.stanford.edu/software/corenlp.shtml#Download

Need to run Stanford CoreNLP tool on the directory of descriptions of images (obtained after running code extract-img-desc.py)
